# Chemical Formulation & Cost Optimization Tool

An open-source Python tool designed for chemical formulation development, specifically tailored for **Paints & Powder Coatings R&D**. This script bridges traditional polymer formulation with data science by automating weight-to-percentage conversions, calculating raw material costs, and generating composition charts.

## Features
- **Formulation Automation:** Input raw material weights and automatically compute chemical percentages.
- **Cost Analysis:** Calculate the total cost per kilogram based on raw material market indices.
- **R&D Optimization Insights:** Embedded logic to flag high-cost formulas and suggest technical adjustments (e.g., filler optimization).
- **Data Visualization:** Built-in Matplotlib integration to output immediate composition charts.

## Technology Stack
- **Language:** Python 3.x
- **Libraries:** Matplotlib (for data visualization)
